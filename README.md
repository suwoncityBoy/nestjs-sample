
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter 

## Installation

```bash
$ npm install
```

Note: TypeORM 사용 시 
### Create new migration
```bash
$ npm run migrate:create <migration name>
```
### Migrate database
```bash
npm run migrate:up
```
## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
